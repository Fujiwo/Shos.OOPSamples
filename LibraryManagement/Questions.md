# 図書管理システム - 問題

## 問題1: 図書管理システムのクラス設計

図書館の蔵書管理システムを作成します。以下の要件を満たすクラス設計を考えてください。

### 要件

1. **書籍 (Book)**: 本の情報（タイトル、著者、ISBN）を管理します
2. **利用者 (Member)**: 図書館の利用者情報を管理します
3. **貸出記録 (LoanRecord)**: 貸出・返却の記録を管理します
4. **図書館 (Library)**: 蔵書と貸出を管理します

### 課題

1. 上記の要件を満たすクラス図を作成してください
2. クラス間の関係（継承、コンポジション、集約など）を明確にしてください
3. 各クラスの主要なプロパティとメソッドを定義してください

### ヒント

- 書籍には、一般書籍、雑誌、DVDなど、さまざまな種類があります。どのように抽象化できるでしょうか？
- 貸出中の本と在庫の本をどのように区別しますか？
- Model（データ管理）とView（画面表示）を分離するにはどうすればよいでしょうか？

## 問題2: プログラムの実装

上記で設計したクラス図を基に、以下の機能を持つプログラムを実装してください。

### 実装する機能

1. 書籍の登録
2. 利用者の登録
3. 書籍の貸出
4. 書籍の返却
5. 蔵書検索
6. 貸出履歴の表示

### 実装言語

- Python
- HTML/CSS/JavaScript

### 注意点

- コードはできるだけシンプルに保ってください
- オブジェクト指向の原則に従ってください
- 各クラスの責務を明確にしてください

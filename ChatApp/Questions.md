# チャットアプリ - 問題

## 問題1: チャットアプリのクラス設計

シンプルなチャットアプリケーションを作成します。以下の要件を満たすクラス設計を考えてください。

### 要件

1. **ユーザー (User)**: チャットに参加するユーザーを表現します
2. **メッセージ (Message)**: 送信されるメッセージを表現します
3. **チャットルーム (ChatRoom)**: メッセージの送受信を管理します
4. **メッセージ履歴 (MessageHistory)**: 過去のメッセージを保存・管理します
5. **表示 (View)**: チャット画面の表示と入力を処理します

### 課題

1. 上記の要件を満たすクラス図を作成してください
2. クラス間の関係（継承、コンポジション、集約など）を明確にしてください
3. 各クラスの主要なプロパティとメソッドを定義してください

### ヒント

- ユーザーとメッセージの関係はどのように表現できるでしょうか？
- メッセージには、テキストメッセージ以外に画像メッセージなども考えられます。どのように抽象化できるでしょうか？
- チャットルームとメッセージ履歴の関係はどうでしょうか？
- Model（チャットデータ）とView（画面表示）を分離するにはどうすればよいでしょうか？

## 問題2: プログラムの実装

上記で設計したクラス図を基に、以下の機能を持つプログラムを実装してください。

### 実装する機能

1. ユーザー登録
2. メッセージの送信
3. メッセージ履歴の表示
4. タイムスタンプの表示
5. ユーザー名の表示
6. メッセージの削除
7. メッセージの検索

### 実装言語

- Python
- HTML/CSS/JavaScript

### 注意点

- コードはできるだけシンプルに保ってください
- オブジェクト指向の原則に従ってください
- 各クラスの責務を明確にしてください
- Model（チャットデータ）とView（画面表示）を分離してください
